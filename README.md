# Calculator

Calculator App using technologies:

• Frontend: ReactJs </br>
• Backend:  node.js

Features: 

• Add </br>
• Subtract </br>
• Multiply </br>
• Divide </br>
• Follows BODMAS rule on multiple operations

Steps to run the application:

Backend:
1. cd CalculatorReact_Server
2. npm install
3. npm start

Frontend:
1. cd CalculatorReact_UI
2. npm install
3. npm start
